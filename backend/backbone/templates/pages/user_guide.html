Backbone User Guide

Welcome to the core integration guide for {{ site_name }}. This document outlines the key functionalities and API endpoints for your frontend application.

Authentication

The authentication system uses JWT-based stateless tokens. All auth endpoints are prefixed with /api/auth.

POST/register: Register a new user. Sends a consolidated welcome/verification email.

POST/login: Authenticate and receive tokens.

POST/password-reset/request: Send a password reset email. Requires email in the body.

Email Templates

Templates are located in backend/backbone/templates/email. You can override them by placing files with the same name in backend/templates/email.

  • welcome_verification.html: Consolidated registration email.
  • order_confirmation.html: Sent after order placement.
  • order_status_update.html: Sent on order status change.

PDF Invoices

Invoices are generated using ReportLab from the HTML template at email/pdf/invoice.html. They are automatically attached to order confirmation emails.

Core Pages

Core pages are served under /pages:

  • /pages/verify-status: Displays verification results.
  • /pages/reset-password/confirm: Password reset form.
  • /pages/user-guide: This guide.
